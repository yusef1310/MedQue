# MedQue

MedQue is a medical laboratory queue management system that allows:
- Patients to book lab tests and receive queue numbers
- Labs to manage daily queues and update appointment status

## Technologies Used
- PHP
- MySQL (PDO)
- Bootstrap 5

## How to Run
1. Import db/database.sql into MySQL
2. Update config.php with database credentials
3. Run using XAMPP or online hosting
